进程与线程区别

简单来说，一个正在运行的程序就是一个进程，例如你电脑上运行的百度网盘，而线程就是网盘里一个个的上传下载任务，一个任务就称为一个线程。一个进程至少拥有一个线程，即主线程，例如Java程序中的main方法。针对不同场景，我们可以编写单线程应用或者多线程应用来满足业务的需求。

多线程通俗例子

举个例子通俗的理解什么是多线程编程。

进程就类比一个口罩生产工厂，线程就类比生产口罩的一条条生产流水线。一个口罩厂至少也有一条生产线去生产口罩，这个针对口罩需求不大的应用场景。当遇到流感季节，口罩需求量大大增加，就需要增加多条生产线同时去生产口罩，以满足人们的使用。

Java实现多线程编程

Java天生就支持多线程编程，创建线程可以通过继承Thread类，或者实现Runnable接口两种方式，启动都需要Thread对象的start()方法来执行，实际工作中推荐使用实现Runnable接口来创建线程，因为在Java中，继承只能一次继承，但可以多次实现接口，可扩展性更强。

  • 继承Thread类

    ThreadDemo01代码

    public class ThreadDemo01 extends Thread {
    public void run() {
        for (int i = 0; i < 100; i++)
            System.out.println(this.getName() + ": " + i + " is running");
    }
}

    ThreadDemo02代码

    public class ThreadDemo02 extends Thread {
    public void run() {
        for (int i = 0; i < 100; i++)
            System.out.println(this.getName() + ": " + i + " is running");
    }
}

    以上定义了两个类，分别都继承了Thread父类，并且重写了run方法，该方法内只是简单的循环打印了一句话，接下来我们在ThreadDemo03中进行测试。

    ThreadDemo03代码

    public class ThreadDemo03 extends Thread {
    public static void main(String[] args) {
        ThreadDemo01 threadDemo01 = new ThreadDemo01();
        threadDemo01.setName("thread01");
        ThreadDemo02 threadDemo02 = new ThreadDemo02();
        threadDemo02.setName("thread02");

        threadDemo01.start();
        threadDemo02.start();
    }
}

    在ThreadDemo03中，我们创建了两个线程threadDemo01和threadDemo02，并执行start方法进行线程启动，运行情况如下所示:

  • 实现Runnable接口

    RunnableDemo01代码

    public class RunnableDemo01 implements Runnable {
    public void run() {
        for (int i = 0; i < 100; i++)
            System.out.println(Thread.currentThread().getName() + ": " + i + " is running");
    }
}

    RunnableDemo02代码

    public class RunnableDemo02 implements Runnable {
    public void run() {
        for (int i = 0; i < 100; i++)
            System.out.println(Thread.currentThread().getName() + ": " + i + " is running");
    }
}

    以上定义了两个类，分别都实现了Runnable接口，并且实现了run方法，该方法内只是简单的循环打印了一句话，接下来我们在RunnableDemo03中进行测试。

    RunnableDemo03代码

    public class RunnableDemo03 {
    public static void main(String[] args) {
        Thread t1 =new Thread(new RunnableDemo01());
        t1.setName("t1");
        Thread t2 =new Thread(new RunnableDemo02());
        t2.setName("t2");

        t1.start();
        t2.start();
    }
}
